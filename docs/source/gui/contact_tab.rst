Contact Tab
===========

Displays:

* **Author** circle with real LinkedIn / GitHub / Gmail / personal-site
  icons (fetched live from the Iconify CDN with a graceful emoji
  fallback).
* **Lab affiliation** (Plant Signaling Lab, IISER Tirupati) with the
  official logos rendered from local PNG / JPEG.
* **Supported databases** (KEGG, NCBI, PlantTFDB, AnimalTFDB,
  MyGene.info) — each linkable circle pulls the live website favicon
  via Google's S2 service.

All circles are clickable and open the relevant URL in the user's
default browser.

Theming
-------

The Contact tab respects the active light/dark theme. Icons recolour
themselves for contrast; the brand colour everywhere is ``#16A085``
(teal).
