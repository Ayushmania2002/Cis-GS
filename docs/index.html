Overview

Cis-GS (Cis-regulatory Element Genome Scanner) is an integrated desktop application developed at the Plant Signaling Lab, IISER Tirupati. It provides a complete analysis pipeline for cis-regulatory element research — from downloading genomes via NCBI to building interactive co-expression networks.

The tool bridges the gap between raw genomic data and actionable regulatory insights by combining promoter extraction, transcription factor binding site (TFBS) search, motif visualization, gene expression integration, and co-expression network analysis in a single, user-friendly application.

Cis-GS can be used in two ways: through a fully graphical interface (GUI) requiring no programming knowledge, or through a command-line interface (CLI) for scripting and automation. A standalone Windows .exe is also available — no Python required.

Get Started

Choose how you want to use Cis-GS:

💾 Windows Standalone Executable — No Python Required

Download the pre-built Cis-GS.exe from GitHub Releases. Double-click to run — no installation, no Python, no dependencies needed.

⇓ Download Cis-GS-v1.0.0-Windows.zip (latest release)
  1. Go to the GitHub Releases page
  2. Under Assets, download Cis-GS-v1.0.0-Windows.zip
  3. Extract the ZIP to any folder on your PC
  4. Double-click Cis-GS.exe to launch
SmartScreen warning: Windows may flag the exe as unrecognised on first launch because it is unsigned. Click "More info" → "Run anyway" to proceed. The software is safe and open source.

GUI Step-by-Step Guide

Launch the GUI with cis-gs (or double-click the .exe). The window opens with eight tabs across the top. Work through them left to right.

Step 0 — Get TF Motifs (Optional but Recommended First)

Before scanning for binding sites, you need motif patterns. Cis-GS integrates three databases. Access them from the Step 2 tab via the "Import from PlantTFDB" or "Import from JASPAR/HOCOMOCO" buttons.

  1. A
    PlantTFDB (plant species)

    In the Step 2 tab, click "Import from PlantTFDB". In the dialog, search for your organism (e.g., "arabidopsis"), select it from the dropdown, choose a save folder, and click "Download from PlantTFDB Server". Once downloaded, switch to the "Browse & Import" tab, filter by TF Family (e.g., MYB, ERF, WRKY), tick the motifs you want, and click "Import Selected → Step 2".

  2. B
    JASPAR / HOCOMOCO (animal/vertebrate species)

    Click "Import from JASPAR/HOCOMOCO" in the Step 2 tab. Select a dataset (e.g., JASPAR 2024 Vertebrates), download it, then browse and filter by species or TF family before importing.

Main Analysis Pipeline

  1. 1
    NCBI Fetch — Download your genome

    Open the NCBI Fetch tab. Type your organism name (e.g., Arabidopsis thaliana) in the search box and press Search. Select an assembly from the results table. Tick Download FASTA and Download GFF3, choose an output folder, and click Download. A progress bar shows download progress and a Stop button lets you cancel.

  2. 2
    Step 1 — Extract Promoter Sequences

    Go to the Step 1 tab. Click Browse to select the downloaded FASTA file, then the GFF3 file. Set the promoter length (default 2000 bp upstream of TSS). Click Extract Promoters. Two output files are saved: a promoter FASTA and a summary TSV table.

  3. 3
    Step 2 — Scan for Motif Hits

    Go to the Step 2 tab. The promoter FASTA from Step 1 should auto-fill. Enter IUPAC motif patterns in the motifs box (one per line, format: NAME<TAB>SEQUENCE), or use the database import buttons. Click Run Scan. Results appear in the table below and are saved as a CSV file.

  4. 4
    Step 3 — Build Sequence Logos

    Go to the Step 3 tab. Select the motif hits CSV from Step 2. Optionally filter by sequence length. Click Generate Logo to produce a publication-quality sequence logo PNG.

  5. 5
    Expression Tab — Link Motif Hits to Expression Data

    Go to the Expression tab. Load your gene expression CSV (rows = genes, columns = samples/time-points). Load the motif hits CSV from Step 2. Click Match Genes. The tool filters the expression matrix to only genes with motif hits and shows a table of results.

  6. 6
    Co-expression Tab — Build a Network

    Go to the Co-expression tab. Load the filtered expression CSV from the Expression tab. Set the correlation method (Pearson or Spearman) and threshold (default 0.7). Click Run Analysis. Outputs: a correlation heatmap, an interactive HTML5 network (opens in your browser), and a module membership CSV.

  7. 7
    K-means Tab — Cluster by Expression Pattern

    Go to the K-means tab. Load an expression CSV. Set K (number of clusters). Click Run K-means. Outputs: cluster assignment CSV, centroid profiles CSV, and a spaghetti plot PNG showing each gene's expression trajectory coloured by cluster.

Tip: Use Settings → Theme in the menu bar to switch between light and dark mode. Your preference is saved automatically.

CLI Step-by-Step Guide

After pip install Cis-GS, the cis-gs command becomes available. Run cis-gs with no arguments to open the GUI, or pass a subcommand to use the analysis pipeline without any graphical interface.

Every command has a built-in --help flag with full details and examples:

cis-gs --help
cis-gs tfdb --help
cis-gs search --help

Step 0 — Get TF Motifs

  1. 0a
    Find your organism's species code (PlantTFDB) 
    cis-gs tfdb species --search arabidopsis
cis-gs tfdb species --search oryza
cis-gs tfdb species                    # list all 157+ species
  2. 0b
    Download motifs for your organism

    Use the 3-letter code from the species list:

    cis-gs tfdb download Ath -o ./motif_db      # Arabidopsis thaliana
cis-gs tfdb download Osa -o ./motif_db      # Oryza sativa
cis-gs tfdb download Zma -o ./motif_db      # Zea mays

    For animal / vertebrate / human studies:

    cis-gs tfdb sources                                         # see all options
cis-gs tfdb download-db jaspar2024_vertebrates -o ./motif_db
cis-gs tfdb download-db hocomoco_human -o ./motif_db
  3. 0c
    Explore and export motifs to a text file 
    # See what TF families are in your downloaded file
cis-gs tfdb filter ./motif_db/Ath_TF_binding_motifs.meme \
       --list-families

# Export all MYB motifs
cis-gs tfdb filter ./motif_db/Ath_TF_binding_motifs.meme \
       --info ./motif_db/Ath_TF_binding_motifs_information.txt \
       --family MYB -o myb_motifs.txt

# Export multiple families (append with >>)
cis-gs tfdb filter ./motif_db/Ath_TF_binding_motifs.meme \
       --info ./motif_db/Ath_TF_binding_motifs_information.txt \
       --family ERF -o erf_wrky.txt
cis-gs tfdb filter ./motif_db/Ath_TF_binding_motifs.meme \
       --info ./motif_db/Ath_TF_binding_motifs_information.txt \
       --family WRKY >> erf_wrky.txt

# Filter by species for JASPAR
cis-gs tfdb filter ./motif_db/JASPAR2024_CORE_vertebrates_non-redundant.meme \
       --species "Homo sapiens" -o human_motifs.txt

    The output motifs.txt is a tab-separated file (NAME<TAB>IUPAC_SEQUENCE) used by cis-gs search.

Step 1 — Download Genome from NCBI

  1. 1
    Fetch FASTA + GFF3 from NCBI using an assembly accession

    Find your assembly accession at ncbi.nlm.nih.gov/assembly (e.g., GCF_000001735.4 for Arabidopsis thaliana TAIR10.1):

    cis-gs fetch GCF_000001735.4 -o ./genome

    This downloads two files into ./genome/:

    ./genome/GCF_000001735.4_genomic.fasta    # genome sequences
./genome/GCF_000001735.4_genomic.gff3    # gene annotations

    Options:

    cis-gs fetch GCF_000001735.4 --no-fasta   # GFF3 only
cis-gs fetch GCF_000001735.4 --no-gff    # FASTA only

Step 2 — Extract Promoter Sequences

  1. 2
    Extract upstream promoter sequences from the genome 
    cis-gs extract ./genome/GCF_000001735.4_genomic.fasta \
               ./genome/GCF_000001735.4_genomic.gff3 \
               -l 1500 -o promoters.fasta

    -l 1500 extracts 1500 bp upstream of each gene's TSS (default is 1000 bp). Outputs: 

    promoters.fasta    # promoter sequences (one per gene)
promoters.tsv      # table of gene IDs, coordinates, strand

Step 3 — Scan for TFBS Motif Hits

  1. 3
    Search promoters for transcription factor binding sites 
    # Using the motifs file exported in Step 0
cis-gs search promoters.fasta --motifs-file myb_motifs.txt -o hits.csv

# Or type motifs directly on the command line
cis-gs search promoters.fasta -m ACGTG -m RGATCY -o hits.csv

# Search forward strand only (skip reverse complement)
cis-gs search promoters.fasta --motifs-file myb_motifs.txt --no-revcomp

    Output hits.csv contains columns: gene_id, motif_name, pattern, strand, position, matched_seq

Step 4 — Build Sequence Logos

  1. 4
    Generate sequence logo PNGs from the motif hit sequences 
    cis-gs logo hits.csv -o ./logos

# Scale options: 'bits' (information content, default) or 'probability'
cis-gs logo hits.csv -o ./logos --scale probability

    One PNG is generated per motif name, saved to ./logos/.

Step 5 — Match Hits with Expression Data

  1. 5
    Filter your expression matrix to genes with motif hits

    Your expression CSV should have genes as rows and samples/time-points as columns (first column = gene IDs):

    cis-gs feed hits.csv expression.csv -o filtered_expr.csv

    Output filtered_expr.csv contains only the genes that had at least one TFBS hit. Gene ID prefixes (gene-, rna-) and version suffixes (.1) are handled automatically.

Step 6a — Co-expression Network Analysis

  1. 6a
    Build a co-expression network with Louvain module detection 
    cis-gs coexpr filtered_expr.csv -o ./coexpr_results

# Use Spearman correlation instead of Pearson
cis-gs coexpr filtered_expr.csv --method spearman -o ./coexpr_results

# Stricter threshold (fewer but stronger edges)
cis-gs coexpr filtered_expr.csv --threshold 0.8 --hide-isolated

    Output files in ./coexpr_results/:

    correlation_matrix.csv         # pairwise correlation values
correlation_heatmap.png        # heatmap image
module_membership.csv          # gene → module assignment
coexpression_network.html      # interactive network (open in browser)
coexpression_network.png       # static network image

Step 6b — K-means Temporal Clustering

  1. 6b
    Cluster genes by expression pattern across time points 
    cis-gs kmeans filtered_expr.csv -k 6 -o ./clusters

    Output files in ./clusters/:

    kmeans_clusters.csv     # gene → cluster assignment
kmeans_centroids.csv    # mean expression profile per cluster
kmeans_plot.png         # spaghetti plot of expression trajectories

    Try different values of -k (number of clusters) and compare the spaghetti plots to find the most biologically meaningful grouping.

Full Example — One-liner Workflow

# Step 0: Download Arabidopsis MYB motifs from PlantTFDB
cis-gs tfdb download Ath -o ./db
cis-gs tfdb filter ./db/Ath_TF_binding_motifs.meme \
       --info ./db/Ath_TF_binding_motifs_information.txt \
       --family MYB -o myb_motifs.txt

# Step 1: Download genome
cis-gs fetch GCF_000001735.4 -o ./genome

# Step 2: Extract 2 kb promoters
cis-gs extract ./genome/GCF_000001735.4_genomic.fasta \
               ./genome/GCF_000001735.4_genomic.gff3 \
               -l 2000 -o promoters.fasta

# Step 3: Scan for MYB binding sites
cis-gs search promoters.fasta --motifs-file myb_motifs.txt -o myb_hits.csv

# Step 4: Build sequence logos
cis-gs logo myb_hits.csv -o ./logos

# Step 5: Filter expression data
cis-gs feed myb_hits.csv expression.csv -o filtered_expr.csv

# Step 6a: Co-expression network
cis-gs coexpr filtered_expr.csv -o ./coexpr_results

# Step 6b: K-means clustering
cis-gs kmeans filtered_expr.csv -k 6 -o ./clusters

Key Features

🧬

NCBI Integration

Search and download FASTA genomes and GFF3 annotations directly via NCBI assembly IDs with real-time progress bar and cancel button.

🔎

Promoter Extraction

Extract upstream promoter sequences from genomic FASTA + GFF3 annotation with configurable promoter length (any number of bp).

🎯

IUPAC Motif Search

Scan promoters for transcription factor binding sites using IUPAC degenerate patterns. Both strands searched by default.

📊

Sequence Logos

Generate publication-quality sequence logos (bits or probability scale) from motif match alignments, filtered by length.

📈

Expression Integration

Match motif hits with gene expression CSV data — automatically handles ID prefix and version differences.

🕸

Co-expression Networks

Correlation-based network with Louvain community detection and interactive HTML5 visualization (Vis.js) with zoom, pan, and module colours.

📉

K-means Clustering

Temporal clustering of gene expression profiles with optimised initialisation and spaghetti plot output.

🗃

Multi-database Motifs

Built-in access to PlantTFDB (157+ plants), JASPAR 2024 (vertebrates & insects), and HOCOMOCO v11 — downloadable directly within the app.

🗺

Full CLI Support

Every GUI feature is also available from the command line — scriptable, automatable, and server-friendly.

🎨

Light & Dark Theme

Modern UI with light/dark mode toggle. Preference saved between sessions.

Supported Motif Databases

🌿 PlantTFDB

  • 157+ plant organisms
  • Filter by TF family (MYB, ERF, WRKY, bHLH…)
  • Download from planttfdb.gao-lab.org
  • Offline fallback catalogue built-in
  • GUI: Step 2 → Import from PlantTFDB
  • CLI: cis-gs tfdb download Ath

🐙 JASPAR 2024

  • Vertebrates: ~575 non-redundant profiles
  • Insects: ~99 non-redundant profiles
  • Filter by species and TF family
  • GUI: Step 2 → Import from JASPAR/HOCOMOCO
  • CLI: cis-gs tfdb download-db jaspar2024_vertebrates

🧬 HOCOMOCO v11

  • Human: ~700 motifs (ChIP-Seq derived)
  • Mouse: ~400 motifs (ChIP-Seq derived)
  • High-quality curated binding models
  • CLI: cis-gs tfdb download-db hocomoco_human
  • CLI: cis-gs tfdb download-db hocomoco_mouse

Data Formats

Input Formats

FormatUsed InDescription
FASTASteps 1–2Genome nucleotide sequences (from NCBI)
GFF3Step 1Gene annotations with coordinates and strand
IUPAC patternStep 2Motif consensus (e.g., GAATTC, RGATCY)
MEMEStep 2Position frequency matrices (PlantTFDB / JASPAR / HOCOMOCO)
CSVExpressionGene expression matrix (genes × samples)

Expression CSV Format

gene_id,0h,6h,12h,24h,48h
AT1G00010,5.2,8.1,6.3,4.9,3.1
AT1G00020,1.1,1.3,1.2,1.0,0.9
AT1G00030,0.5,2.8,7.4,6.1,2.3

First column = gene IDs. Remaining columns = samples or time-points.

Motifs File Format (for CLI)

# Lines starting with # are comments
MYB|AT1G00010  AACCGTTA
ERF|AT2G00020  GCAGCCGCC
WRKY|AT3G00030 TTGACY

Tab-separated: NAME<TAB>IUPAC_SEQUENCE per line.

IUPAC Ambiguity Codes

R = A or G     Y = C or T     S = G or C     W = A or T
K = G or T     M = A or C     B = C, G, or T  D = A, G, or T
H = A, C, or T V = A, C, or G N = any base
